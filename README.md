# Gemini Desktop

Десктопное приложение на базе Electron для удобной работы с **Google Gemini** и экосистемой сервисов **Google Workspace**.

> ⚠️ **Дисклеймер:** Данное приложение является неофициальным open-source клиентом и не имеет никакого отношения к компании Google.

## Основные возможности

*   🌐 **Обход региональных ограничений (Smart DNS):**
    *   Встроенный DNS-резолвер с поддержкой DoH (`xbox-dns.ru`) и проксирование через адреса `111.88.96.50` / `111.88.96.51` для беспрепятственного доступа к Gemini.
*   ✨ **Интерактивные световые волны (Travelling Edge Glow):**
    *   При переключении вкладок по периметру экрана запускается плавная световая волна с заполняющей неоновой нитью, адаптированная под цветовую гамму каждого сервиса (фиолетовый для Gemini, красный для Gmail, цветной для Поиска и т.д.).
*   🎨 **Прозрачная заставка запуска (Aurora Splash):**
    *   Минималистичная заставка при запуске приложения: только логотип-звезда Gemini и мягкое вращающееся свечение Авроры.
*   🖥️ **Прозрачный оверлей с поддержкой Click-Through:**
    *   Возможность закрепить приложение поверх всех окон с регулируемой прозрачностью.
    *   **Шорткат `Ctrl+Shift+L`:** переводит окно в сквозной режим (мышь кликает сквозь приложение на окна сзади), позволяя использовать Gemini как полупрозрачный виджет поверх рабочего стола.
*   🎛️ **Хаб сервисов Google (Google Services Hub):**
    *   Встроенная сетка быстрого доступа к популярным веб-сервисам: **Gmail, Google Поиск, NotebookLM, YouTube, Google Переводчик и Google Календарь**.
    *   Удобная кнопка **«Домой»** для мгновенного возврата на домашнюю страницу Gemini или в меню выбора сервисов.
*   🔒 **Обход блокировок авторизации:**
    *   Подмена User Agent (Firefox 128) для безопасного входа в учетные записи Google без ошибок «небезопасного браузера».
*   📥 **Интеграция с ОС:**
    *   Кастомный фрейм окна без рамок (frameless window).
    *   Сворачивание в системный трей при закрытии приложения.

## Шорткаты и управление

| Сочетание клавиш | Описание |
| :--- | :--- |
| `Ctrl+Shift+L` | Переключение сквозного клика (Click-Through) в режиме закрепленного поверх окон оверлея. |
| **Кнопка «Закрепить» 📌** | Переключение режима поверх всех окон (Always on Top) с активацией прозрачности. |
| **Кнопка «Домой» 🏠** | Возврат к Gemini или сброс к сетке выбора сервисов. |

## Установка и запуск

### Готовая сборка

*   [Release](https://github.com/KiwiProject-V-NT/Gemini-Desktop/releases)


### Требования
*   Установленный [Node.js](https://nodejs.org/) (рекомендуется LTS версия).

### Шаги для запуска в режиме разработки
1.  Клонируйте или скачайте репозиторий с исходным кодом.
2.  Откройте терминал в папке проекта и установите зависимости:
    ```bash
    npm install
    ```
3.  Запустите приложение:
    ```bash
    npm start
    ```

## Сборка EXE (Компиляция)

Для создания независимого портативного исполняемого файла для Windows выполните:

```bash
npm run build
```

После завершения процесса в папке `dist/` появится готовый файл:
`dist/Gemini Desktop 1.0.0.exe`

*Сборка является портативной (portable), не требует установки и содержит все необходимые зависимости.*

## Конфигурация

Все пользовательские настройки (выбранный режим DNS, закрепление окон, прозрачность) сохраняются автоматически в файле `config.json` в рабочей директории приложения (App Data пользователя):
`%APPDATA%/gemini-desktop-app/config.json`

Пример структуры `config.json`:
```json
{
  "dnsMode": "doh",
  "alwaysOnTop": false,
  "overlayOpacity": 0.9,
  "window": {
    "width": 1200,
    "height": 800,
    "isMaximized": false
  }
}
```
